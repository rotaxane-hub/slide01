1日目：　2/11(木) 東京 → 札幌

詳細は下スライド

[ 11:00 - 12:00 ] 成田空港(第1ターミナル)集合
バーコードでチェックインして発券。荷物預ける
時間あれば軽くお昼を食べる
[ 13:45 - 15:35 ] 飛行機 成田空港 → 新千歳空港
新千歳空港ちょっとブラリしても良い
[ - 17:00 ] 新千歳空港 → 札幌駅
快速エアポートが15分毎に運行しているのでそれに乗る
宿の無料送迎バスは時間がギリギリ過ぎるため使わない
チェックイン時刻変更を宿に連絡
[ - 20:00 ] 札幌駅周辺
荷物をロッカーに預けて散歩・買い物・ご飯
朝食ここで買った方が良いかも（宿からコンビニまで徒歩11分）
[ 20:00 - 20:30 ] 札幌駅 → 真駒内駅
南北線で真駒内駅へ移動（乗車時間30分）
or バスで宿付近へ（20:00発 → 21:07第一ホテル前バス停着）
or 宿までタクシー使う（料金：約8,200円、時間：約1時間）
[ - 21:30 ] 真駒内駅 → 第一ホテル前バス停
じょうてつバスで移動（20:56発 → 21:35第一ホテル前バス停着）
or 宿までタクシー使う（料金：約5,700円、時間：約35分）
[ - 21:45 ] 第一ホテル前バス停 → 宿
バス停から宿（翠山亭倶楽部定山渓）まで徒歩10分
チェックイン時に、10:20発の送迎バス（帰りの分）を予約する

3日目：　2/13(土) 札幌 → 函館

詳細は下スライド

[ 10:00 - ] チェックアウト・移動
10:20発の送迎バスで札幌へ移動
[ 11:30 - ] 札幌でラーメン食べたい
どこのお店行くか道中調べようね

[ 13:27 - 16:55 ] スーパー北斗で函館へ

4日目：　2/14(日) 函館 → 東京

詳細は下スライド
